# Mise à jour

Le système installé utilise `pacman` et Snapper. Vérifier les annonces Arch,
créer un snapshot avant une mise à jour importante, appliquer `pacman -Syu`,
redémarrer si le noyau ou systemd change, puis vérifier Niri, le réseau et le
démarrage de secours. Les snapshots ne remplacent pas les sauvegardes.
